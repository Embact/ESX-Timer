# timer

That's my first Project in ESX  
it's just **Timer Counter**    

# Install  
1. Download the .zip.
2. Extract the .zip or Open the .zip.
3. Place ``timer`` in your ESX Directory
4. Add start ``timer`` to your ``server.cfg``

# Use

**Type** in **Chat:** ``/timer [YOURTIME]``<hr>
example : **/timer 5**<br>
